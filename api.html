1. Tecnologías de Web APIs

Una API (Interfaz de Programación de Aplicaciones) es un conjunto de reglas y protocolos que permiten la comunicación entre sistemas de software. Facilitan la interoperabilidad entre aplicaciones distribuidas mediante estándares como HTTP y formatos de datos como JSON o XML.

Principios REST

  • Arquitectura cliente-servidor
  • Comunicación sin estado (stateless)
  • Uso de métodos HTTP
  • Representación de recursos mediante URLs

Métodos HTTP

  • GET: Obtener recursos
  • POST: Crear recursos
  • PUT: Actualizar recursos
  • DELETE: Eliminar recursos
