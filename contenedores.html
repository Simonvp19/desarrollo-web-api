4. Manejo de Contenedores

¿Qué es un contenedor?

Un contenedor es una unidad ligera y portátil que empaqueta una aplicación junto con todas sus dependencias, garantizando que se ejecute de manera uniforme en distintos entornos.

Docker

Docker es una plataforma que permite crear, distribuir y ejecutar contenedores. Facilita el desarrollo, pruebas y despliegue continuo de aplicaciones.

Kubernetes

Kubernetes es un sistema de orquestación que administra múltiples contenedores, permitiendo escalabilidad automática, balanceo de carga y alta disponibilidad.

Relación con Microservicios

Cada microservicio puede ejecutarse dentro de un contenedor independiente, lo que permite su despliegue y escalabilidad de forma autónoma en entornos de nube.
