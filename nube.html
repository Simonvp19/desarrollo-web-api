5. Desarrollo de Aplicaciones en la Nube

La computación en la nube es un modelo de prestación de servicios tecnológicos que permite el acceso bajo demanda a recursos informáticos como servidores, almacenamiento, bases de datos y plataformas de desarrollo a través de Internet.

Modelos de Servicio

Modelo Descripción Ejemplo
IaaS Infraestructura como servicio. Proporciona servidores virtuales y redes. AWS EC2
PaaS Plataforma como servicio. Permite desarrollar y desplegar aplicaciones. Heroku, Azure App Service
SaaS Software como servicio. Aplicaciones listas para usar. Google Workspace

Modelos de Despliegue

  • Nube pública: Recursos compartidos vía Internet.
  • Nube privada: Infraestructura exclusiva para una organización.
  • Nube híbrida: Combinación de pública y privada.

Principales Proveedores

  • Amazon Web Services (AWS)
  • Microsoft Azure
  • Google Cloud Platform

Ventajas del Desarrollo en la Nube

  • Escalabilidad automática
  • Alta disponibilidad
  • Reducción de costos iniciales
  • Despliegue continuo
  • Integración con contenedores y microservicios

Relación con APIs y Microservicios

Las APIs desarrolladas bajo arquitectura REST pueden desplegarse en entornos cloud para garantizar disponibilidad global. Asimismo, los microservicios se ejecutan comúnmente dentro de contenedores Docker orquestados con Kubernetes en plataformas de nube pública.
